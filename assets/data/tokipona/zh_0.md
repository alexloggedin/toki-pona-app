对不起，我在学习中文，我的普通话不很好。

语言”toki pona” （你会说没有商调：太开破那）只有14个罗马字的字。都toki pona的字说这道：a，e，i，j, k, l, m, n, o, p, s, t, u, w.
这字都是小写。
说字的字头：

| toki pona |中文的概语音|日本的|
|:---------:|:----------:|:----:|
| a         |啊          |「あ」|
| e         |诶          |「え」|
| i         |一          |「い」|
| j         |元          |      |
| k         |开          |      |
| l         |了          |      |
| m         |猫          |      |
| n         |你          |      |
| o         |噢          |「お」|
| p         |票          |      |
| s         |三          |      |
| t         |图          |      |
| u         |鱼（的韵）  |「う」|
| w         |我          |      |

都tokipona字强调一号音节。

这tokipona字来英文。

* mi 来 me
* kulupu 来 group
* lukin 来 looking

这tokipona字来中文。

* sinpin 来粤语的 “前邊”
* kon 来普通话的 “空气”
* sin 来普通话的 “新”
